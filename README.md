# 🚀 PixelPass - Gerador de Senhas

**Objetivo do Sistema:** O PixelPass é uma aplicação web focada em segurança e praticidade. O sistema permite ao usuário gerar senhas aleatórias e criptograficamente seguras com base no tamanho desejado. Além de gerar, a plataforma armazena a finalidade da senha (ex: "E-mail Corporativo") e a data de criação em um banco de dados relacional, mantendo um histórico acessível para consulta através de uma interface amigável.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python
* **Framework Web:** Flask
* **Banco de Dados:** SQLite (via Flask-SQLAlchemy)
* **Formulários e Segurança:** Flask-WTF e WTForms (proteção CSRF)
* **Front-end:** HTML5 (Jinja2), CSS3 (Metodologia BEM) e SweetAlert2 para notificações
* **Controle de Versão:** Git e GitHub

## ⚙️ Instruções de Execução

Para rodar este projeto na sua máquina local, siga os passos abaixo:

1. Clone o repositório:
   git clone [https://github.com/luisfernandogrizoni/projeto-integrador.git](https://github.com/luisfernandogrizoni/projeto-integrador.git)
2. Entre na pasta do projeto:
   cd projeto-integrador
3. Instale as bibliotecas necessárias
   pip install flask flask-sqlalchemy flask-bcrypt flask-wtf
4. Execute a aplicação a partir do arquivo principal:
   python main.py
